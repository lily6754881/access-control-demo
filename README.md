# 门禁管理系统原型

## 系统概述

这是一个基于Web的门禁管理系统原型，采用现代化的UI设计，包含完整的门禁管理功能模块。

## 功能模块

### 1. 系统概览 (index.html)
- 系统状态总览
- 设备统计信息
- 快速操作入口
- 最近活动记录

### 2. 门禁监测 (monitoring.html)
- 电子地图：多层级地图展示（园区/楼栋/楼层/区域/点位）
- 门禁控制：单个设备控制和批量控制操作
- 状态监测：实时状态监控和报警处理
- 门禁点位可视化和设备状态实时显示
- 地图联动控制和搜索筛选功能
- 控制日志记录和报警统计图表

### 3. 设备台账 (records.html)
- 设备台账管理
- 事件记录查询
- 报警记录管理
- 统计报表导出

### 4. 权限管理 (permission.html)
- 用户管理
- 角色管理
- 权限配置
- 组织管理
- 操作审计

### 5. 系统设置 (settings.html)
- 参数配置
- 设备接入
- 备份恢复
- 安全设置
- 系统信息

## 技术特点

- **响应式设计**: 支持桌面和移动设备
- **现代化UI**: 使用Tailwind CSS构建
- **交互体验**: 丰富的动画效果和用户反馈
- **模块化架构**: 各功能模块独立开发
- **统一导航**: 一致的页面切换体验

## 文件结构

```
原型文件/
├── index.html          # 主页面（系统概览）
├── monitoring.html     # 门禁监测页面（包含电子地图、门禁控制、状态监测）
├── records.html        # 设备台账页面
├── permission.html     # 权限管理页面
├── settings.html       # 系统设置页面
└── README.md           # 说明文档
```

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 使用左侧导航菜单切换不同功能模块
3. 各页面支持完整的交互操作
4. 所有页面都采用统一的UI风格和交互模式

## 设计特色

- **白色底色**: 简洁明快的视觉风格
- **简明设计**: 突出功能性和易用性
- **状态指示**: 清晰的状态颜色编码
- **实时反馈**: 操作结果即时显示
- **数据可视化**: 图表和统计信息展示
